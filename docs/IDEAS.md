# Идеи на будущее (gen-memento-docs)

Сюда можно складывать **сырые идеи**: фичи, контент, рефакторинг, эксперименты. Когда идея созреет — вынести в отдельный план в `docs/superpowers/plans/` (и при необходимости спеку в `docs/superpowers/specs/`).

## Как вести список

- Одна строка или короткий абзац на идею; по желанию — тег: `[контент]`, `[dev]`, `[wiki]`, `[ux]`.
- Можно добавлять дату: `2026-03-31 — …`
- Не обязательно приоритизировать; приоритет можно отметить вручную: `(?)` сомнительно, `(!)` важно.
